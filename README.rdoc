== README

* Ruby version

  Ruby ruby 2.3.0p0 (2015-12-25 revision 53290)

* Installation

  Please run "bundle install" and "rake db:migrate" to setup the system

  Please run "rails s" and open browser to connect "http://localhost:3000/"

  You can import events json by click "Import new events" or link to "http://localhost:3000/events/new"


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
